# Homebrew Tap

Custom Homebrew formulas.

## Installation

```bash
brew tap minorole/tap
```

## Formulas

| Formula | Description |
|---------|-------------|
| `gsx` | Ghostty Session Manager - launch AI-ready dev environments |

### gsx

```bash
brew install minorole/tap/gsx
```

Or after tapping:

```bash
brew install gsx
```
